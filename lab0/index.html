HTML - HYPERTEXT MARKUP LANGUAGE

HTML, or HyperText Markup Language, is the standard, foundational language used to create and structure web pages. It acts as the skeleton for all websites, using a system of tags and elements to define content, such as headings, paragraphs, images, and links for web browsers. It is not a programming language, but a markup language that determines how content is organized on a page.

TAGS DESCRIPTION
<head> Contains meta information about the document like title, styles, and scripts.
<title> Defines the title of the web page shown on the browser tab.
<body> Contains all the visible content of the web page.
<p> Defines a paragraph of text.
<br> Inserts a single line break.
<hr> Creates a horizontal line to separate content.
<img> Embeds an image in the web page.
<a> Creates a hyperlink to another page or resource.
<ul> Defines an unordered (bulleted) list.
<ol> Defines an ordered (numbered) list.
<li> Defines a list item inside ul or ol.
<table> Creates a table to organize data.
<tr> Defines a row in a table.
<td> Defines a data cell in a table.
<th> Defines a header cell in a table.

HTML FORM TAGS

List of form tags

  • <form> - Defines an HTML form for user input
  • <input> - Defines an input control
  • <textarea> - Defines a multiline input control (text area)
  • <button> - Defines a clickable button
  • <select> - Defines a drop-down list
  • <option> - Defines an option in a drop-down list
