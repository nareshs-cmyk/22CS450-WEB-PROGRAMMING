HTML - HYPERTEXT MARKUP LANGUAGE

HTML, or HyperText Markup Language, is the standard, foundational language used to create and structure web pages. It acts as the skeleton for all websites, using a system of tags and elements to define content. It is a markup language that determines how content is organized on a page.

Core HTML Tags

TAG DESCRIPTION
<head>Contains meta information about the document.
<title>Defines the title shown on the browser tab.
<body>Contains all the visible content of the page.
<p>Defines a paragraph of text.
<br>Inserts a single line break.
<hr>Creates a horizontal line separator.
<img>Embeds an image in the web page.
<a>Creates a hyperlink to another resource.
<ul>Defines an unordered (bulleted) list.
<ol>Defines an ordered (numbered) list.
<table>Creates a table to organize data.

HTML FORM TAGS

Common Form Elements:

  • <form> - The container for user input.
  • <input> - General purpose input control.
  • <select> - A dropdown selection list.
  • <button> - A clickable action button.
